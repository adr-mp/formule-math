Dérivées

Dérivée de sin(x) : f'(x) = cos(x)

Dérivée de cos(x) : f'(x) = -sin(x)

Dérivée de e^x : f'(x) = e^x

Dérivée de ln(x) : f'(x) = 1/x

Dérivée de tan(x) : f'(x) = 1/cos²(x)

Dérivée de x^n : f'(x) = nx^(n-1)

Primitives

Primitive de cos(x) : F(x) = sin(x) + C

Primitive de sin(x) : F(x) = -cos(x) + C

Primitive de 1/x : F(x) = ln|x| + C

Primitive de e^x : F(x) = e^x + C

Primitive de x^n : F(x) = (x^(n+1))/(n+1) + C

Séries

Série géométrique : Σ r^n = 1/(1-r), |r| < 1

Série de Taylor e^x : e^x = 1 + x + x²/2! + x³/3! + ...

Suite de Fibonacci : Fₙ = Fₙ₋₁ + Fₙ₋₂

Série harmonique : Σ 1/n = 1 + 1/2 + 1/3 + ...

Série de Maclaurin sin(x) : sin(x) = x - x³/3! + x⁵/5! - ...

Probabilités et Statistiques

Espérance : E[X] = Σ (x * P(x))

Variance : Var(X) = E[X²] - (E[X])²

Loi normale : f(x) = (1/σ√(2π)) * e^(-(x-μ)²/2σ²)

Covariance : Cov(X,Y) = E[(X-E[X])(Y-E[Y])]

Coefficient corrélation : ρ = Cov(X,Y)/(σ_X * σ_Y)

Loi binomiale : P(X=k) = C(n,k) * p^k * (1-p)^(n-k)

Identités Trigonométriques

Pythagore : sin²(x) + cos²(x) = 1

Double angle sin : sin(2x) = 2sin(x)cos(x)

Double angle cos : cos(2x) = cos²(x) - sin²(x)

Addition sin : sin(A+B) = sin(A)cos(B) + cos(A)sin(B)

Addition cos : cos(A+B) = cos(A)cos(B) - sin(A)sin(B)

Tangente : tan(x) = sin(x)/cos(x)

Géométrie

Aire cercle : A = πr²

Volume sphère : V = (4/3)πr³

Surface sphère : S = 4πr²

Volume cylindre : V = πr²h

Volume cône : V = (1/3)πr²h

Théorème de Pythagore : a² + b² = c²

Algèbre

Équation quadratique : x = (-b ± √(b² - 4ac))/(2a)

Binôme de Newton : (x + y)^n = Σ C(n,k)x^(n-k)y^k

Identité remarquable 1 : (a + b)² = a² + 2ab + b²

Identité remarquable 2 : (a - b)² = a² - 2ab + b²

Identité remarquable 3 : (a + b)(a - b) = a² - b²

Calcul Différentiel

Règle du produit : (uv)' = u'v + uv'

Règle du quotient : (u/v)' = (u'v - uv')/v²

Règle de la chaîne : (f∘g)' = (f'∘g) * g'

Dérivée seconde : f''(x) = (f'(x))'

Dérivée partielle : ∂f/∂x = lim(h→0) [f(x+h,y) - f(x,y)]/h

Intégration

Intégrale définie : ∫[a→b] f(x)dx = F(b) - F(a)

Intégration par parties : ∫udv = uv - ∫vdu

Changement de variable : ∫f(x)dx = ∫f(g(t))g'(t)dt

Intégrale double : ∫∫D f(x,y)dxdy

Théorème de Green : ∮C (Pdx + Qdy) = ∫∫D (∂Q/∂x - ∂P/∂y)dxdy

Matrices

Déterminant 2x2 : |A| = ad - bc

Matrice inverse 2x2 : A⁻¹ = 1/|A| * [[d,-b],[-c,a]]

Trace : tr(A) = a₁₁ + a₂₂ + ... + aₙₙ

Produit matriciel : (AB)ᵢⱼ = Σₖ aᵢₖbₖⱼ

Transposée : (Aᵀ)ᵢⱼ = aⱼᵢ
